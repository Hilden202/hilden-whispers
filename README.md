# 🎙️ Hilden Visioner

En minimalistisk podcastplattform byggd med React + Vite för Hilden Media.  
Projektet fokuserar på enkelhet, atmosfär och direkt ljuduppspelning utan onödigt brus.

</div>

---

## 🌌 Om projektet

Hilden Visioner är ett experiment i att kombinera:

- teknik
- berättande
- ljud
- design
- atmosfär

…i ett eget litet digitalt poddrum.

Projektet fungerar som både:
- podcastspelare
- PWA
- frontend-experiment
- kreativ plattform för framtida avsnitt

---

## ✨ Funktioner

- 🎧 Inbyggd ljudspelare
- 📱 Mobilanpassad design

## 🚀 Lokal utveckling

Klona projektet:

```bash
git clone https://github.com/Hilden202/hilden-whispers.git
```

Installera dependencies:

```bash
npm install
```

Starta utvecklingsserver:

```bash
npm run dev
```

Build:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

---

## 🎵 Ljudfiler

Podcastavsnitt placeras i:

```txt
/public/audio
```

Exempel:

```txt
/public/audio/Hilden Podd Avsnitt 1.mp3
```

MP3 rekommenderas för webben:
- mindre filstorlek
- bättre streaming
- bättre mobilstöd
