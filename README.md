# 🎙️ Hilden Visioner

En atmosfärisk podcastplattform byggd med React + Vite för Hilden Media.

Projektet kretsar kring utforskande samtal om tid, verklighet, information och medvetande — presenterat genom AI-genererade röster, tankeexperiment och minimalistisk design.

</div>

---

## 🌌 Om projektet

Hilden Visioner började som ett experiment:

Vad händer om man kombinerar programmering, AI, ljuddesign och filosofiska tankeexperiment i ett och samma projekt?

Resultatet blev en minimalistisk podcastplattform där avsnitten utforskar frågor om:

- tid
- perception
- information
- observatörer
- medvetande
- verklighetens natur

Projektet fungerar både som:

- podcastplattform
- frontend-experiment
- kreativt AI-projekt
- PWA-applikation
- digitalt idéarkiv

Allt paketerat i Hilden Medias mörka och atmosfäriska designspråk.

---

---

## 🧠 Ursprung och idéer

Podcasten bygger delvis på egna tankeexperiment, modeller och visioner kring tid, information, perception och verklighetens natur.

Flera av idéerna och modellerna har tidigare publicerats och dokumenterats via Zenodo som ett personligt och kreativt utforskningsprojekt.

---

## 🎙️ Produktion

Röstsegmenten genererades med hjälp av ElevenLabs och kombinerades därefter manuellt med atmosfäriska ljudmiljöer och ljuddesign i Logic Pro.

Projektet är ett experiment i gränslandet mellan:

- AI-genererad media
- programmering
- ljuddesign
- filosofi
- digital storytelling

## 🚀 Lokal utveckling

Klona projektet:

```bash
git clone https://github.com/Hilden202/hilden-whispers.git
```

Installera dependencies:

```bash
npm install
```

Starta utvecklingsserver:

```bash
npm run dev
```

Build:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

---

## 🎵 Ljudfiler

Podcastavsnitt placeras i:

```txt
/public/audio
```

Exempel:

```txt
/public/audio/Hilden Podd Avsnitt 1.mp3
```

MP3 rekommenderas för webben:
- mindre filstorlek
- bättre streaming
- bättre mobilstöd
